Motive

The motive behind this project is to protect internet users from deceptive and manipulative practices employed by some websites. These dark patterns can lead to unintended purchases, subscriptions, and the unwarranted sharing of personal information. Additionally, the project aims to enhance overall cybersecurity by providing protection against phishing attacks and other online threats.

Goals

The primary goal is to develop a comprehensive tool that offers multiple layers of protection and user awareness:

  • Identify and alert users about dark patterns on websites.
  • Protect users from phishing attacks.
  • Enhance cybersecurity measures through VPN integration.
  • Provide a user-friendly multi-platform tool, including a browser extension and a desktop/mobile application.

Initial Mini Project

In the initial phase, the project will focus on developing a web extension and a basic web portal with the following functionalities:

  • Detection of dark patterns on websites.
  • Basic phishing protection features.
  • User alerts and educational resources about online safety.

This phase aims to create a working prototype to demonstrate the core functionalities and gather user feedback for further improvements.

Future Developments

Building on the initial mini project, future developments will include:

  • Enhanced phishing detection and prevention mechanisms.
  • Comprehensive cybersecurity features, including VPN services.
  • Development of a desktop application using PyQt.
  • Development of a mobile application using Kivy.
  • Regular updates and maintenance based on user feedback and emerging threats.

These developments aim to provide a robust and versatile tool that caters to the diverse needs of internet users across different platforms.
